All design documents are currently named and grouped by project.

gcrewmet_design contains the LoggerNet design for the gcrewmet15min table.

GENX_design contains the LoggerNet design for genx_ardlog, genx_control, genx_export, and genx_redoxswap.
GENX_plotnames contains plot names and experimental design for GENX.

logger_x_project contains table / logger names and which project folders they are paired with.

organs_design contains the LoggerNet design for the control and export tables of the GENX organs experiment.
organs_plotnames contains plot names and experimental design for the GENX organs experiment.

redox_design contains the LoggerNet design for the table1 and tableambient tables for the C3 SMARTX experiment.
redox_plotnames contains the plot names experimental design for the redox part of the C3 SMARTX experiment.

site_plotnames contains the basic site information for site-level data tables (e.g., gcrewmet_gcrewmet15min, all waterlevel tables, and gcrewmarshoutlet_sontektable).

SMARTX_design contains the LoggerNet design for c3log_ and c4log_control and export, as well as the design for co2log_co2log.
SMARTX_plotnames contains plot names and experimental design for c3log, c4log, and co2log.

sontektable_design contains the LoggerNet design for the gcrewmarshoutlet_sontektable table.

waterlevel_design contains the LoggerNet design for all waterlevel tables (genx, gcrewmet, c3log, c4log).
